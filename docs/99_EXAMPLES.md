# 例文集

## 文頭

```
誰が雌豚だよ、てめえときたら
そんなことよりもまずは
今年の夏も猛暑が予想される中、
昨今の情勢を鑑みるに、
論より証拠と言うけれども、
```

## 文中

```
なんて余裕ぶってみたものの、
やっぱり借金って怖いもので、
忘れてしまった子供たちときたら、悪びれもせず
以来、数多くの文明が栄えては滅びてきたが、
考慮しなければならないのは、
```

## 文末

```
セルフレジに関しては完全に諦めたんです。
けれど、そんなことはどうでもいい。
生命保険を見直すことにした。
だなんて、歯切れの悪い様子だった。
彼はただ泣きそうな顔で笑った。
```
